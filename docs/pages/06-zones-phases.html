Phase System

Phases let you reveal architecture in layers — like building blueprints. A slider appears in the header when phases are defined.

1
Skeleton
Core API, database, key services — always visible
2
Processing
Queues, caches, workers — appear at phase 2+
3
Observability
CDN, monitoring, dashboards — appear at phase 3
Visibility rule: An item is visible when its phase index ≤ the selected phase index. Items without a phase field are always visible. Array-valued phases allow items to appear and then disappear at later phases.
