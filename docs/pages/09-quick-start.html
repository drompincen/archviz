9 Quick Start

Get ArchViz running in 3 steps

Option A: jbang (single command)

# Install jbang (if needed)
$ curl -Ls https://sh.jbang.dev | bash -s - app setup
# Run ArchViz
$ jbang ArchViz.java
# Open http://localhost:8080/collab-animation.html

Option B: Maven

# Run directly
$ mvn spring-boot:run
# Or build and run the JAR
$ mvn package -DskipTests
$ java -jar target/archviz-0.2.0.jar
Prerequisites
✓ Java 17+
DynamoDB is optional — defaults to in-memory storage
URL Parameters
collab=<file>.jsonLoad a specific diagram
story=trueAuto-activate story mode
Adding Your Own Diagrams
Drop a .json file into src/main/resources/static/json/ — it auto-appears in the dropdown. See pages 4–8 for the JSON format.
Tip: Use ?collab=coffee-shop-transformation.json&story=true to jump straight into the most feature-rich example with full story mode.
