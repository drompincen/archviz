2 Features Overview

Six integrated views from a single JSON definition

Spatial View
Drag-and-drop nodes on a canvas with animated message particles. Zones draw infrastructure boundaries. Connections show data flow paths.
Sequence View
Auto-generated sequence diagram from the same JSON. Step-through mode lets you pause and advance one step at a time.
Story Mode
Wizard-style narrative: Problem → Vision → Roadmap phases. Slide navigation with keyboard arrows and dot indicators.
Benefits Panel
Phase-grouped benefit cards that accumulate as you advance. Click a card to highlight the bound nodes for 3 seconds.
KPI Dashboard
Live KPI HUD that updates per phase. Tracks numeric and qualitative metrics with color-coded direction indicators.
PDF Export
Export spatial view, sequence diagram, notes, and logs to PDF. Respects the active phase for point-in-time snapshots.

Additional Capabilities
